In this project I build a monthly calender. The calender shows the current date, the whole month and which day out of the week and month it is.

I used HTML, CSS and Javascript for this project. My challenge was to always get the current date displayed and here I had to use Javascript. I made a function for the day, the month and the year to get the current date. In those functions I simply call the current date from the webbrowser by using the function 'new date'.
